# Overview

This is a barebones webapp for tracking health and turnorder for table top RPG games like Dungeons and Dragons. I created it so I could learn about django and how to use it to make interactive webapps and because it seems like something that could be useful ot me persoanlly in my own hobbies. I only have a little bit of experience using things like javascript in web development and previous to this I had never done anything with databases so I learned a lot about that. Although the webapp is not fully functional, I learned a lot about forms and how they work.

[Software Demo Video](https://youtu.be/Ku2PgkwI6KQ)

# Web Pages

One page is a combat tracker that allows you to modify the health of characters in the encounter. The other page allows you to see all of the characters you have created and create new ones.

# Development Environment

This was developed in VS Code using python and the django library.
